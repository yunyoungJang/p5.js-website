description: >
  <p>색상의 밝기 값을 가져옵니다.</p>

  <p><code>lightness()</code> 함수는 

  <a href="/reference/p5/p5.Color">p5.Color</a> 객체, 색상 구성 요소의 배열,
  또는 CSS 색상 문자열에서

   HSL 색상의 밝기 값을 추출합니다.</p>

  <p>기본적으로, <code>lightness()</code> 함수는 색상의 HSL 밝기를 
  0부터

  100까지의 범위로 반환합니다. 만약 <a href="/reference/colorMode">colorMode()</a> 가 HSL로 설정되어 있다면,
  주어진 범위 내의

  밝기 값을 반환합니다.</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(50);

      // HSL 색상을 사용하세요.
      colorMode(HSL);

      // HSL 값을 사용하여 p5.Color 객체를 생성합니다.
      let c = color(0, 100, 75);

      // 왼쪽 직사각형을 그립니다.
      noStroke();
      fill(c);
      rect(15, 15, 35, 70);

      // 밝기 값 ('lightValue') 을 75로 설정합니다.
      let lightValue = lightness(c);

      // 오른쪽 직사각형을 그립니다.
      fill(lightValue);
      rect(50, 15, 35, 70);

      describe('직사각형 두 개.왼쪽은 살몬 핑크이고, 오른쪽은 회색입니다.');
    }
    </code>
